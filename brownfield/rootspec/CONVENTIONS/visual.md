# Visual Conventions

## Color Palette
- **Primary gradient:** #667eea → #764ba2 (used on active states, weather card, forecast bars)
- **Background:** #f0f2f5 (app background)
- **Surface:** white (cards, chips, panels)
- **Border:** #ddd (default), #667eea (active/hover)
- **Text primary:** #333
- **Text secondary:** #666, #999
- **Error:** #e53e3e on #fff5f5
- **Alert colors:** red (#fed7d7/#c53030), amber (#fefcbf/#975a16), blue (#bee3f8/#2b6cb0), gray (#e2e8f0/#4a5568)

## Typography
- **Font stack:** -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
- **App title:** 24px, color #333
- **Section headings (h3):** 18px, color #333
- **Body:** 14px
- **Small/labels:** 12-13px, color #666/#999

## Spacing & Layout
- **App max-width:** 640px (weather view), 1024px (comparison-active)
- **App padding:** 20px
- **Card padding:** 24px, border-radius 16px
- **Chip padding:** 6px 14px, border-radius 20px
- **Button padding:** 6-8px 12-16px, border-radius 6-8px

## Component Visuals
- **Weather card:** gradient background (#667eea → #764ba2), white text, large temperature (64px/700w)
- **Favorites chips:** white background, active state uses primary gradient color as background
- **Settings panel:** white surface, toggle button labeled "⚙️ Settings"
- **Compare button:** white bg, #667eea border and text
- **Dashboard grid:** auto-fill columns, 180px min, 12px gap
