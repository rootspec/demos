# Visual Conventions

## Color Palette
- **Primary:** RootSpec brand blue (#2563eb)
- **Secondary:** Complementary accent colors
- **Neutral:** Gray scale for text and backgrounds
- **Success:** Green for positive actions
- **Warning:** Amber for attention

## Typography
- **Headings:** System font stack with fallbacks
- **Body:** Readable font size (16px base)
- **Code:** Monospace for technical content
- **Scale:** Consistent type scale (1.25 ratio)

## Layout
- **Container:** Max-width with centered content
- **Grid:** CSS Grid for complex layouts
- **Flexbox:** Flex for component-level layout
- **Responsive:** Mobile-first breakpoints
- **Spacing:** Consistent spacing scale (0.25rem base)

## Interactive Elements
- **Buttons:** Clear primary/secondary hierarchy
- **Links:** Underlined with hover states
- **Forms:** Clear labels and validation states
- **Focus:** Visible keyboard focus indicators

## Theme System
- **Light/Dark:** Dual theme support
- **CSS Variables:** Theme-aware custom properties
- **System Detection:** Respects user preference
- **Manual Override:** Toggle with persistence
- **Smooth Transitions:** Non-jarring theme changes
