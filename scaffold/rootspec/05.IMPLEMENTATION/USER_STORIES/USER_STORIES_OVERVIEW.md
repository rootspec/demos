# User Stories Overview

**References:** All spec levels (L1–L4)

---

## Story Organization

Stories are organized by phase and grouped by system.

### MVP Phase

All stories are in the MVP phase — the full experience as described in the SEED.md.

| File | System Coverage | Stories |
|------|----------------|---------|
| meta.yaml | VIEW_SYSTEM | Meta banner, footer, navigation |
| feed.yaml | FEED_SYSTEM, DATA_SYSTEM | Home feed, pagination, like, bookmark, composer |
| profile.yaml | PROFILE_SYSTEM | Profile view, follow/unfollow |
| thread.yaml | FEED_SYSTEM, DATA_SYSTEM | Post detail, thread view, replies |
| search.yaml | DISCOVERY_SYSTEM | Keyword search, empty state |
| discovery.yaml | DISCOVERY_SYSTEM | Explore page, tag filtering, user suggestions |

---

## Story ID Convention

- `US-001` through `US-009` — META / VIEW_SYSTEM stories
- `US-010` through `US-029` — FEED_SYSTEM stories
- `US-030` through `US-039` — PROFILE_SYSTEM stories
- `US-040` through `US-049` — Thread / post detail stories
- `US-050` through `US-059` — DISCOVERY_SYSTEM / Search stories
- `US-060` through `US-069` — DISCOVERY_SYSTEM / Explore stories
