# Visual Conventions

## Theme System
- **Dark Theme:** Primary theme with dark backgrounds and light text
- **Light Theme:** Secondary theme with light backgrounds and dark text
- **Toggle:** Theme switcher available in header for user preference
- **Persistence:** Theme choice persists across sessions

## Color Palette
- **Primary Background:** Dark gray (#1a1a1a) / Light gray (#f8f9fa)
- **Secondary Background:** Medium gray (#2d2d2d) / White (#ffffff)
- **Text Primary:** Light gray (#e5e5e5) / Dark gray (#1a1a1a)
- **Text Secondary:** Medium gray (#a0a0a0) / Medium gray (#6b7280)
- **Accent:** Blue (#3b82f6) for interactive elements
- **Border:** Subtle gray (#404040) / Light gray (#e5e7eb)

## Typography
- **Font Family:** System font stack (system-ui, sans-serif)
- **Heading Sizes:** text-xl (20px) for main headings, text-lg (18px) for subheadings
- **Body Text:** text-base (16px) for primary content
- **Small Text:** text-sm (14px) for secondary content and metadata
- **Weight:** font-medium (500) for headings, font-normal (400) for body

## Layout Patterns
- **Container:** max-width with horizontal padding for content areas
- **Cards:** Subtle borders with rounded corners for post cards and content blocks
- **Spacing:** Consistent margin and padding using Tailwind's spacing scale
- **Grid:** Responsive layouts using CSS Grid and Flexbox

## Interactive Elements
- **Buttons:** Rounded corners with hover states and focus outlines
- **Links:** Blue accent color with hover effects
- **Forms:** Clear borders with focus states for inputs
- **Icons:** Consistent size and color for action buttons (like, bookmark, follow)

## Component States
- **Active States:** Visual feedback for liked, bookmarked, and followed items
- **Hover Effects:** Subtle background changes for interactive elements
- **Loading States:** Skeleton screens or loading indicators for async content
- **Error States:** Clear error messages with appropriate styling
